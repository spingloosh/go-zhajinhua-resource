# go-zhajinhua-resource

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面以静态资源形式存储，不针对任何特定域名或网站，仅作为通用资源存放与展示使用。

## 目录说明

```
.
├── pages/          # 存放所有独立的 HTML 页面文件
├── assets/         # 页面引用的静态资源（如 CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/` 目录下每个 HTML 文件均为一个独立页面，可直接在浏览器中打开查看。
- `assets/` 目录存放页面所需的公共资源文件，按需引用。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务或数据库。
- 页面内容仅用于资料归档与展示，不包含任何指向外部网站或服务的链接。
- 页面文件名及目录结构可能会根据归档需求进行调整，建议以最新提交为准。

## 维护说明

- 本仓库不定期更新，新增或修改页面时会在提交记录中说明。
- 如发现页面显示异常或资源缺失，欢迎提交 Issue 反馈。
- 不接受针对具体网站或商业用途的定制请求。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件即可查看。
3. 如需引用 `assets/` 中的资源，请确保路径正确。

## 许可证

本项目内容仅供学习与归档参考。
